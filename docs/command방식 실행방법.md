# remove_password.py 사용법

## 설치

```bash
pip install pikepdf
```

---

## 출력 파일명 규칙

원본 파일은 절대 수정하지 않습니다. 비밀번호가 제거된 사본을 `_open` 접미사로 생성합니다.

```
원본: 개인진료정보내역(기본진료정보)_20260413.pdf
출력: 개인진료정보내역(기본진료정보)_20260413_open.pdf
```

---

## 1. 비밀번호를 아는 경우

### 단일 파일

```bash
python remove_password.py "data/개인진료정보내역(기본진료정보)_20260413.pdf" --password 12345678
python remove_password.py "data/개인진료정보내역(처방조제정보)_20260413.pdf" --password 12345678
python remove_password.py "data/개인진료정보내역(세부진료정보)_20260413.pdf" --password 12345678
```

### 폴더 전체 (한 번에)

```bash
python remove_password.py data/ --password 12345678
```

**출력 예시:**
```
총 3개 파일 처리 시작 (크기 오름차순)

  [OK] 완료 | 원본 217.0KB → 출력 217.7KB (100.3%) | 개인진료정보내역(기본진료정보)_20260413_open.pdf
  [OK] 완료 | 원본 330.1KB → 출력 331.0KB (100.3%) | 개인진료정보내역(처방조제정보)_20260413_open.pdf
  [OK] 완료 | 원본 709.6KB → 출력 710.9KB (100.2%) | 개인진료정보내역(세부진료정보)_20260413_open.pdf

완료: 3개 성공, 0개 실패
```

---

## 2. 비밀번호를 모르는 경우 (브루트포스)

숫자 8자리를 자동으로 탐색합니다.  
**날짜 패턴(생년월일 YYYYMMDD)을 먼저 시도**하므로, 건강보험공단 등 공공기관 문서는 보통 수십 초 내 완료됩니다.

### 단일 파일

```bash
python remove_password.py "data/개인진료정보내역(기본진료정보)_20260413.pdf" --brute-force
python remove_password.py "data/개인진료정보내역(처방조제정보)_20260413.pdf" --brute-force
python remove_password.py "data/개인진료정보내역(세부진료정보)_20260413.pdf" --brute-force
```

### 폴더 전체 (한 번에)

```bash
python remove_password.py data/ --brute-force
```

**출력 예시:**
```
스마트 브루트포스: 개인진료정보내역(기본진료정보)_20260413.pdf
  1단계: 날짜 패턴(YYYYMMDD, 1940~2010) 시도 중...
[OK] 비밀번호: 19630713 | 14.3초 (8594번 시도) | 저장: 개인진료정보내역(기본진료정보)_20260413_open.pdf
```

### 소요 시간 안내

| 상황 | 예상 시간 |
|---|---|
| 생년월일 패턴 (공공기관 문서) | 수 초 ~ 수십 초 |
| 날짜 패턴 전체 탐색 (1940~2010) | 최대 2~3분 |
| 전체 브루트포스 (00000000~99999999) | 최대 수 시간 |

---

## 옵션 요약

| 옵션 | 설명 |
|---|---|
| `--password 숫자8자리` | 비밀번호 직접 지정 |
| `--brute-force` | 자동 탐색 (날짜 패턴 우선) |
| `--output 폴더경로` | 출력 위치 지정 (기본: 원본과 동일 폴더) |
